# TASK_03_FILTER_PIPELINE

Scope:
Visual filters and post-processing only.

Goal:
Transform visuals from “technical demo” into premium cinematic glossy visuals.

Target:
- rich color
- premium contrast
- glossy highlights
- cinematic bloom
- controlled sharpening
- depth
- readability

Avoid:
- muddy image
- dirty grading
- oversaturation
- blur soup
- cheap chromatic aberration

Inspect current filter chain before modification.

Prefer pipeline:

1. source
2. composition
3. color grading
4. contrast
5. highlight enhancement
6. bloom
7. final clarity
